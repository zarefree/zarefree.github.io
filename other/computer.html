Visual Studio Code 配置 ​

  • 插件目录软连接

    提示

    插件目录默认位置无法修改，只能设置软链接以免插件占用系统盘资源

  1. 剪切C:\Users\xxx\.vscode至D:\Documents\.vscode
  2. 以管理员身份运行命令提示符
  3. mklink /D C:\Users\xxx\.vscode D:\Documents\.vscode
